\### HymmnosReader



\##### Overview

---



A simple program written in C# designed to manage and sort the entire Hymmnos lexicon from the Ar Tonelico series.



Features:

&nbsp;- Includes a UTF-16-encored .txt file that contains the entire Hymmnos lexicon

&nbsp;- Displays characteristics of the Hymmnos language, such as the number of records in each Hymmnos dialect

&nbsp;- Sorts the lexicon alphabetically based on the word, meaning, class, kana, or dialect

&nbsp;- Allows the user to search for specific word parameters such as:

&nbsp;    - Nouns

&nbsp;    - Verbs

&nbsp;    - Adjectives

&nbsp;    - Prepositions

&nbsp;    - Conjunctions

&nbsp;    - Adverbs

&nbsp;    - Pronouns

&nbsp;    - Constructs

&nbsp;    - Particles

&nbsp;    - Interjections

&nbsp;    - Emotion-Verbs

&nbsp;    - Emotion-Sounds

&nbsp;- Includes a toggle to remove New Testament of Pastalie words from the search results; as Pastalie is a dialect unique to the Infel Phira SH server, it cannot be used with other Hymmnos words.
